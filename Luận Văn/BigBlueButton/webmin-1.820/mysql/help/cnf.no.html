MySQL Server konfigurasjon
Denne siden kan brukes til å justere globale konfigurasjonsinnstillinger for MySQL. Du vil som regel ikke trenge å endre noen av disse, selv om buffer og cache størrelser kan økes på systemer med stort antall tabeller og data.

For at endringene skal tre i kraft må MySQL tjeneren stoppes og omstartes, vha. knappene på modulens hovedside.
