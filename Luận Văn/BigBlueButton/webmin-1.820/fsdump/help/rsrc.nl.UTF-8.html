Herstellen vanaf een file of een apparaat
Deze optie bepaald vanwaar de backup dump data zal worden gelezen. Dit kan een lokale file zijn of een tape apparaat, of een file of apparaat op een remote host. Voordat het herstellen van een remote host kan werken moet het rmt protocol worden ondersteund of aanstaan. Deze staat normaal gesproken standaard niet aan om veiligheid's redenen.
