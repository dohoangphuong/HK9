Añadir Campo
En esta página puedes añadir un nuevo campo a la tabla. Los parámetros editables para el nuevo campo son:
Nombre de campo
El nombre de este campo en la tabla.

Medida de tipo (para campos varchar y char)
El máximo número de caracteres permitidos para datos en este campo.

¿Campo de arreglo?
Si se pone esta opción a Sí, el campo será capaz de almacenar valores del mismo tipo.

Al terminar, haz click en el botón Salvar para actualizar la estructura de la tabla con el nuevo campo.
