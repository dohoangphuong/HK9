PostgreSQL-Benutzer
Diese Seite ermöglicht Ihnen das Erstellen und Bearbeiten von Benutzern, die Zugang zu Ihren PostgreSQL-Datenbanken haben. Für jeden Benutzer können Sie einen Benutzernamen, ein optionales Passwort, ein optionales Verfallsdatum wählen. Zusätzlich können Sie wählen, ob es den Benutzer erlaubt sein soll andere Nutzer oder Datenbanken zu erzeugen.
