Accés Lectura/Escriptura
Això determina quins hosts tenen accés de lectura/escriptura a aquest servidor. Hi ha tres tries per aquesta opció:
  • Cap
    No hi ha cap host amb accés de lectura/escriptura.
  • Tots els hosts
    Qualsevol host pot muntar aquest recurs lectura/escriptura. Si estàs connectat/da a Internet, això vol dir que qualsevol persona del món pot llegir, modificar i esborrar els fitxers del directori exportat. Utilitza aquesta opció amb molta cura.
  • Hosts Llistats
    Els hosts o adreces IP de la llista de sota poden muntar aquest recurs lectura/escriptura. Els hosts de la llista de sota poden muntar aquest recurs lectura/escriptura. Els hosts es poden especificar de les formes següents:

    Nom de host o adreça IP (com ara ftp.foo.com o 1.2.3.4)
    Coincideix amb el host del mateix nom o adreça
    Grup de xarxa (com ara enginyers)
    Coincideix amb qualsevol host membre del grup de xarxa
    Domini DNS (com ara .foo.com)
    Coincideix amb qualsevol host del domini
    Xarxa (com ara @10.254.1)
    Coincideix amb qualsevol host de la xarxa
    Xarxa/màscara (com ara @10.254.1/24)
    Coincideix amb qualsevol host de la xarxa

    A més a més, totes les notacions anteriors poden prefixar-se amb -, que vol dir que el host, grup de xarxa, domini o xarxa no té permès de muntar el recurs. Això pot ser útil per permetre l'accés a tot un grup de hosts, però denegant-lo a un dels membres.
