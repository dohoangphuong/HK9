Acceso de Lectura-Escritura
Esto determina qué máquinas tienen acceso de lectura-escritura a esta compartición. Hay tres elecciones para esta opción:
  • Ninguna
    Ninguna máquina tiene acceso de sólo-lectura.
  • Todas las máquinas
    Cualquier máquina puede montar esta compartición como de sólo-lectura. Si estás conectado a Internet, esto quiere decir que cualquiera a nivel mundial puede leer archivos de este directorio exportado.
  • Máquinas listadas
    Las máquinas o direcciones IP listadas debajo pueden montar esta compartición como de sólo-lectura. Las máquinas que coincidan con las listadas debajo pueden montar esta compartición como de sólo-lectura. Las máquinas pueden ser especificadas de la siguiente manera:

    Nombre de máquina o dirección IP (como ftp.foo.com o 1.2.3.4)
    Que coincida la máquina del mismo nombre o dirección
    Grupo de Red (como ingeniería)
    Que coincida cualquier máquina que sea miembro del grupo de red
    Dominio DNS (como .foo.com)
    Que coincida cualquier máquina del dominio
    Red (como @10.254.1)
    Que coincida cualquier máquina de la red
    Red/máscara de red (como @10.254.1/24)
    Que coincida cualquier máquina de la red

    Además, cualquiera de las anotaciones de arriba puede llevar el prefijo -, queriendo esto decir que la máquina, grupo de red, dominio o red no tiene permisos para montar esta compartición. Esto puede ser útil si desea permitir el acceso a un conjunto de máquinas pero denegárselo a un miembro.
