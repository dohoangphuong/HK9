Accès Lecture Seule
Ceci détermine quels hôtes à l'accès en lecture seule sur ce partage. Trois choix vous sont offert :
  • Aucun
    Aucun hôte ne peut avoir l'accès en lecture.
  • Tous les hôtes
    Tous les hôtes peuvent monter ce partage en lecture. Si vous êtes connecté à Internet, ceci veut dire que n'importe qui dans le monde peut lire les fichiers du répertoire exporté.
  • Hôtes Listés
    Les hôtes listés ci-dessous peuvent monter ce partage en lecture seulement, les hôtes peuvent être spécifier de cette façon :

    Nom d'hôte ou adresse IP (comme ftp.foo.com ou 1.2.3.4)
    Correspond aux hôtes avec le nom ou l'adresse
    Groupe réseau (comme enginering)
    Correspond à tout hôte qui sont membre de ce groupe réseau
    Nom de domaine (comme .foo.com)
    Correspond à tout hôte dans ce domaine
    Réseau (comme @10.254.1)
    Correspond de tout hôte dans ce réseau
    Réseau/masque de sous-réseau (comme @10.254.1/24)
    Correspond à tous hôtes dans ce réseau

    De plus, toutes les notations ci-dessus peuvent commencer par un -, ce qui signifie que l'hôte, domaine ou réseau n'est pas autorisé à monter ce partage. Ceci peut être utile pour accepter l'accès à une série d'hôte tout en le refusant à un membre.
