LDAP Server
Met deze module kunt u de OpenLDAP directory server configureren en de objecten beheren in zijn database. Ervan uitgaande dat u een LDAP server heeft geinstalleerd op hetzelfde systeem als waar Webmin geinstalleerd is. De hoofd pagina laat iconen zien waarmee u de server configuratie kunt bewerken en de database mee kunt beheren. Indien de module zo is geconfigureerd dat u een remote LDAP server beheerd zal alleen iconen laten zien om toegang te krijgen tot de database.

U kunt bepalen of het een lokale of een remote server moet beheren door te klikken op de Module Config link. Alleen wanneer u een lokale server beheerd zult u een pagina krijgen om configuratie files te bewerken en om het OpenLDAP server proces te stoppen, starten en te herstarten.

Indien u alleen Unix gebruikers en groepen wilt beheren in uw LDAP database moet u kijken bij de LDAP Gebruikers en Groepen module die u vind in de Webmin Systeem categorie. Wilt u het systeem configureren om gebruikers en groepen op te halen van een lokale of remote LDAP server kunt u het beste de LDAP Client module gebruiken, ook deze module vind u in de Webmin Systeem categorie.
