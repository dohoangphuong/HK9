File Sets
Een file set is een lijst met files en directory's die Bacula kan backuppen als onderdeel van een taak. Iedere set moet een unieke naam hebben, een lijst met files om in te voegen en een optionele lijst met files die worden uitgesloten. Deze functie kan handig zijn om niet critische of directory's over te slaan die zich bevinden in een diectory die word uitgesloten.
