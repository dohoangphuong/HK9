Volume Pools
In de Bacula context is een pool een set met volumes die bestemd zijn voor een bepaald doel. Een volume is normaal gesproken een bepaalde tape of bestemmings directory, waarbij ieder is gelabeld met een unieke naam. Wanneer u een backup taak maakt kunt u selecteren welke pool zal worden gebruikkt als bestemmings file en dus welk volume.

Een volume kan worden toegevoegd aan een pool met gebruik van de Volume Label pagina, gemarkeerd met een tape of bestemmings directory met een unieke naam.
