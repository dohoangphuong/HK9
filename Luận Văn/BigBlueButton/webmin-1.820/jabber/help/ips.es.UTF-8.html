Conrtol de Acceso por IP
En esta página puede controlar qué direcciones IP tienen permiso para acceder a su servidor Jabber. Si se introduce alguna dirección en la sección IP de cliente permitidas, entonces sólo se concederá permiso a esas direcciones - el resto de direcciones no tendrán permiso para conectar. Sin embargo, cualquier dirección IP listada en la sección IP de cliente denegadas será bloqueada, tanto si aparece en la lista de direcciones permitidas como si no. Ambos campos aceptan direcciones IP (como 1.2.3.4) o parejas de IP/máscara (como 1.2.3.0/255.255.255.0) para especificar redes enteras.
