Control de Tráfico Karma
El servidor Jabber tiene la capacidad de restringir el ancho de banda que puede usar un cliente, que se puede configurar usando las opciones de esta página :
  • Manejo de conexiones excesivas
    Si está activado, el servidor sólo permitirá que un cliente haga un limitado número de conexiones durante un periodo de tiempo dado.

  • Modo Karma
    Para esta opción puede elegir que el servidor no fuerze los límites de tráfico por cliente, elegir una de las tres configuraciones predeterminadas de límites (bajo, medio o alto), o introducir los límites que seleccione. Sólo debería usar esta última opción si está familiarizado con los significados de los campos.
