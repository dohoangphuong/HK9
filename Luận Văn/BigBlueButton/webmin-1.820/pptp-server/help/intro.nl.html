PPTP VPN Server
Met deze module kunt u een PoPToP server opzetten zodat uw systeem PPTP verbindingen kan accepteren van cliënten. Op de hoofdpagina vind u iconen voor algemene instellingen voor de PPTP server opties (zoals de IP adressen die toegekend moeten worden aan cliënten). Voor het instellen van PPP opties (zoals welk soort authenticatie u gaat gebruiken), Voor het bewerken van PPP accounts die authenticatie cliënten bevatten en voor het bekijken van actieve PPTP sessies.

Op de hoofdpagina bevinden zich ook knoppen om de PPTP server te starten of te stoppen. Wanneer de server aan het werk is zal er ook een Configuratie Toevoegen knop te zien zijn om de huidige PPTP server opties te activeren voor nieuwe verbindingen. Alle wijzigingen voor de PPP opties of accounts zullen onmiddellijk worden toegevoegd aan alle nieuwe verbindingen die vanaf dat moment verbinden met de server zonder dat u op deze knop hoeft te drukken.
