Bases de Recerca LDAP
La base de dades d'un servidor LDAP està organitzada jeràrquicament, típicament amb diferents branques en l'arbre per a usuaris, grups, i altres serveis. Aquesta pàgina permet seleccionar on cal buscar a la base de dades LDAP els diferents tipus d'informació, i amb quina profunditat per a cadascun.

Per defecte, totes les recerques es faran sota la Base de recerca global. No obstant, com que la majoria de servidors posen els usuari i grups sota subarbres diferents, probablement hauràs de configurar la Base dels Usuaris Unix i la Base dels grups Unix amb els noms d'aquests subarbres. La Base de les contrasenyes Unix s'ha d'establir amb el mateix DN que la Base dels usuaris Unix, ja que la informació de les contrasenyes gairebé sempre s'emmagatzema juntament amb altres atributs d'usuari.
